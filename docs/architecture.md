# Architecture

High-level overview of the organ controller system architecture.

## Components

- **Input Layer**: CAN bus console, USB-MIDI, mock inputs
- **Logic Layer**: Stop management, couplers, pistons, manual routing
- **Output Layer**: MIDI to ranks, recording, diagnostics
- **Control Plane**: REST API for monitoring and configuration
- **State Management**: Centralized organ state with event dispatching
